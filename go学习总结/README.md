### 为什么要学习Go
1. 支持并发，这是Go与生俱来的特性
2. 简单易学，与C语言类似，学过Java的也推荐学习，性能可以于C比肩
3. 静态类型语言，静态类型的语言就是可以在编译的时候检查出来隐藏的大多数问题
4. 跨平台编译
5. 内嵌C支持

### Go可以做些什么
1. 网络编程，开发Web应用、API应用等
2. 内存数据库，前一段时间Google开发的groupcache，couchbase的部分组建
3. 云平台，目前国外很多云平台在采用Go开发
