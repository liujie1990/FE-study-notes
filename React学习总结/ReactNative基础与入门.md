React Native是facebook开源的一套用于开发移动端跨平台App的技术框架。
### 1. 为什么要用React Native
#### 1.1  传统开发的痛点
1. 人员稀缺
2. 开发成本高
3. 代码复用率低
4. 无法动态更新

#### 1.2  React Native的优点
1. 跨平台(支持开发ios和安卓)
2. 低投入高回报
3. 性能高
4. 支持动态更新
5. 代码复用率高(ios和安卓中代码基本一致)
